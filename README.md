# בין לב לארי 💙

דף נחיתה מקצועי לקליניקה לעיסויים בבאר שבע

## אודות הפרויקט

דף נחיתה מעוצב ומודרני עבור "בין לב לארי" - קליניקה לעיסויים במחירים סטודנטיאליים בבאר שבע.
הקליניקה מנוהלת על ידי אהרון לב ארי, סטודנט שנה רביעית לפסיכותרפיה.

## תכונות

- 🎨 עיצוב מודרני עם צבעים חמים ומרעננים
- ✨ אנימציות ואפקטים חלקים
- 📱 תמיכה מלאה במובייל (Responsive)
- 💬 המלצות לקוחות
- 📝 טופס יצירת קשר אינטראקטיבי
- 📍 פרטי קשר ומיקום

## טכנולוגיות

- HTML5
- React 18
- Tailwind CSS
- Babel (JSX)

## איך להריץ

פשוט פתח את הקובץ `index.html` בדפדפן:

```bash
# דרך 1: פתיחה ישירה
open index.html

# דרך 2: עם שרת מקומי
npx serve .
```

## עריכה והתאמה אישית

### שינוי צבעים
הצבעים מוגדרים ב-CSS classes:
- `gradient-bg` - רקע סגול-סגלגל
- `warm-gradient` - גרדיאנט חם (ורוד)
- `refresh-gradient` - גרדיאנט רענן (כחול)

### עדכון תוכן
כל התוכן נמצא בקובץ `index.html`:
- המלצות לקוחות: `testimonials` array
- פרטי קשר: בסקשן `Location & Contact`

### חיבור לשרת
הטופס כרגע לא שולח נתונים לשרת. לחיבור אמיתי:
1. החלף את פונקציית `handleSubmit`
2. שלח את הנתונים ל-API/Database
3. אפשרויות: Supabase, Firebase, או backend מותאם אישית
